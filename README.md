# Data-Vacc-Covid-19-Turki
Kaggle project
https://www.kaggle.com/omercolakoglu/turkey-covid-19-vaccination-data

ID,	DATE_,	SEQID,	CITY,	CITY2,	_1DOSE,	_2DOSE,	_TOTAL,	POPULATION,	DIFF_1DOSE,	DIFF_2DOSE,	DIFF_TOTAL, and	PREVID.
